Sizes

Control the size of a Y gate using the data-size attribute. 

<y-gate data-size="xs"></y-gate>
<y-gate data-size="sm"></y-gate>
<y-gate data-size="base"></y-gate>
<y-gate data-size="lg"></y-gate>
<y-gate data-size="xl"></y-gate>

Disabled Gate

To display the disabled Y gate, set the data-disabled attribute. 

<y-gate data-disabled></y-gate>

Connecting with Other Gates

To connect the Y gate to the quantum gate of the lower bit, set the data-wire-top attribute. Similarly, set data-wire-bottom for the upper bits. 

<y-gate data-wire-top></y-gate>
<y-gate data-wire-bottom></y-gate>
<y-gate data-wire-top data-wire-bottom></y-gate>
<y-gate data-wire-top-disabled></y-gate>
<y-gate data-wire-bottom-disabled></y-gate>
<y-gate data-wire-top-disabled data-wire-bottom-disabled></y-gate>

To show disabled connections, use the data-wire-top-disabled and data-wire-bottom-disabled attributes. 

<y-gate data-wire-top-disabled></y-gate>
<y-gate data-wire-bottom-disabled></y-gate>
<y-gate data-wire-top-disabled data-wire-bottom-disabled></y-gate>
<y-gate data-wire-top-disabled data-disabled></y-gate>
<y-gate data-wire-bottom-disabled data-disabled></y-gate>
<y-gate data-wire-top-disabled data-wire-bottom-disabled data-disabled></y-gate>

IF directive

To apply a Y gate conditionally, specify a data-if attribute; the condition (variable name) specified in the if directive can be set in the Measurement gate. 

<y-gate data-if="y"></y-gate>

If the gate is connected to another gates, the position of the if label will be adjusted automatically.
