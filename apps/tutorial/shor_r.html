周期 \(r\) を求めるには、パターンを取り出すツールである QFT を使います。 たとえば \(N = 15\), \(a = 2\) の場合、後で実際に実行して確認しますが、 計算した \(2^x\,mod(15)\) に QFT を適用し測定すると、確率のピークがいくつか得られます。

確率のピークは、周期 \(r\) のヒントを与えるものです。 ピークの数はちょうど周期 \(r\) と同じになり、0 から始まって等間隔に並びます。

測定すると 0, 4, 8, 12 のうち 1 つがランダムに手に入るので、測定結果からは直接周期 \(r\) は得られません。 しかし、これらは等間隔でありまた x の範囲は 0 〜 15 と分かっていることから、周期 \(r\) の候補が推定できます。

この \(r\) の候補を出すアルゴリズムを連分数アルゴリズムと呼びます。 詳細は省きますが{% nmargin_note cfa %}興味のある読者は、Qiskit チュートリアル「ショアのアルゴリズム」を参照してください。https://qiskit.org/textbook/ja/ch-algorithms/shor.html{% endnmargin_note %} これは古典コンピュータで計算でき、\(N = 15\), \(a = 2\) の場合には \(r\) の候補として 4 と 8 が得られます。

あとは \(r = 4, 8\) それぞれの候補について \(gcd(N, a^{r/2} + 1)\) と \(gcd(N, a^{r/2} - 1)\) を計算して、それぞれが \(N = 15\) の因数になっているかチェックします。 \(r = 4\) の場合因数 3 と 5 がみつかるので、計算は終了です。
