複数の量子ビットを使うと、たくさんの状態を表すことができます。 たとえば 2 量子ビットを並べると、それぞれの量子ビットは {% ket 0 %} および {% ket 1 %} の状態を持つことができるため、その組合わせは次の 4 通りになります。

  • {% ket 0 %} と {% ket 0 %}
  • {% ket 0 %} と {% ket 1 %}
  • {% ket 1 %} と {% ket 0 %}
  • {% ket 1 %} と {% ket 1 %}

スペースを節約するために、{% ket 0 %} と {% ket 0 %} のように量子ビットを並べたものを {% ket 00 %} とまとめて書くことにしましょう。 すると、先ほどの 4 通りは次のように書けます。

  • {% ket 0 %} と {% ket 0 %} → {% ket 00 %}
  • {% ket 0 %} と {% ket 1 %} → {% ket 01 %}
  • {% ket 1 %} と {% ket 0 %} → {% ket 10 %}
  • {% ket 1 %} と {% ket 1 %} → {% ket 11 %}

ここで、書き直した中身の数字を 2 進数として見てみます。 すると、先ほどの 4 通りはさらに次のように書き換えることができます。

  • {% ket 0 %} と {% ket 0 %} → {% ket 00 %} (2 進数) → {% ket 0 %} (10 進数)
  • {% ket 0 %} と {% ket 1 %} → {% ket 01 %} (2 進数) → {% ket 1 %} (10 進数)
  • {% ket 1 %} と {% ket 0 %} → {% ket 10 %} (2 進数) → {% ket 2 %} (10 進数)
  • {% ket 1 %} と {% ket 1 %} → {% ket 11 %} (2 進数) → {% ket 3 %} (10 進数)

つまり、2 量子ビット使うと {% ket 0 %}, {% ket 1 %}, {% ket 2 %}, {% ket 3 %} という 4 つの状態を表すことができます。 これは、Qni の状態ベクトル表示では次のように 4 つの円のうち、どれか 1 つを塗り潰した状態として表せます。

{{ 0 | qubit_circle: 0, 'lg' }} {{ 1 | qubit_circle: 1, 'lg' }} {{ 0 | qubit_circle: 2, 'lg' }} {{ 0 | qubit_circle: 3, 'lg' }}

さらに 3 量子ビットに増やすと円の数はどうなるでしょうか? 1 量子ビットでは円が 2 つ、2 量子ビットでは円が 4 つだったので、単純に円 6 個と考えるのは間違いです。 先ほどと同様に 3 桁の 2 進数を考えると、3 量子ビットでは {% ket 0 %} から {% ket 7 %} まで 8 通りの状態を表すことができます。 つまり、3 量子ビットを状態ベクトルとして表示すると丸が 8 個になります。

{{ 0 | qubit_circle: 0, 'lg' }} {{ 0 | qubit_circle: 1, 'lg' }} {{ 0 | qubit_circle: 2, 'lg' }} {{ 0 | qubit_circle: 3, 'lg' }} {{ 0 | qubit_circle: 4, 'lg' }} {{ 1 | qubit_circle: 5, 'lg' }} {{ 0 | qubit_circle: 6, 'lg' }} {{ 0 | qubit_circle: 7, 'lg' }}

このようにビット数を増やすと表せる状態 (数) が倍々になっていく現象は、実は古典ビットでもまったく同じです。 古典ビットは 0 と 1 の値を持つので、3 桁の 2 進数で考えると 0 から 7 まで 8 通りの数を表せるからです。 しかし複数の量子ビットでは、次に紹介するように 1 量子ビットの場合と同じく重ね合わせ状態を持つことができるという大きな違いがあります。
