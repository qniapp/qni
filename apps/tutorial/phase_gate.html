PHASE ゲートはその名の通り{% nmargin_note phase-gate %}phase: 相、位相、フェーズ{% endnmargin_note %}、位相を操作するゲートです。 重ね合わせ状態にある量子ビットに適用することで、指定した角度だけ {% ket 1 %} の位相を回転できます。

{% include minus.svg class="h-6 w-6 fill-current text-gray-400" %} {% include arrow_right.svg class="h-6 w-6 fill-current text-gray-400" %}
{% include minus.svg class="h-6 w-6 fill-current text-gray-400" %} {% include arrow_right.svg class="h-6 w-6 fill-current text-gray-400" %}

PHASE ゲートは角度を示すパラメータを必ず 1 つ持ちます。 たとえば、位相を π/2 回転する PHASE ゲートは上に π/2 が付きます。文中では PHASE(π/2) などとパラメータを付けて書きます。

PHASE ゲートは {% ket 0 %} に何の変化も与えないことに注意してください。 また PHASE ゲートは {% ket 1 %} の位相のみを回転するので、振幅の大きさにも影響は与えません。また回転する角度は振幅の大きさの影響を受けません{% nmargin_note phase-and-amplitude-norm %}量子ビットの振幅の大きさと位相は独立したパラメータである、ということを思い出しましょう (状態ベクトル表示を参照)。{% endnmargin_note %}。

{% include minus.svg class="h-6 w-6 fill-current text-gray-400" %} {% include arrow_right.svg class="h-6 w-6 fill-current text-gray-400" %}

マイナスの角度を指定することで、位相を逆方向に回転できます。 たとえば PHASE(π/2) に対するマイナス回転の PHASE(-π/2) を作ることができます。 位相を π/2 回転した後 -π/2 すると元に戻るので、 PHASE(π/2) と PHASE(-π/2) はお互いに逆演算です。

ハンズオン

PHASE ゲートを重ね合わせ状態に適用して、{% ket 1 %} の位相を回転してみましょう。 PHASE ゲートの角度はデフォルトで π/2 にセットされています。変更するには、回路上に配置した PHASE ゲートをクリックし、ポップアップメニューから角度のアイコンをクリックしてください。
