--- layout: page title: 重ね合わせ状態 description: 複数量子ビットでの重ね合わせ ---

1 量子ビットの場合と同じく、複数量子ビットでも重ね合わせ状態を取ることができます。 ここでは 3 量子ビットのいろいろな状態を考えてみましょう。

次の状態では {% ket 2 %} が確率 100% なので、3 量子ビットすべてを測定すると必ず {% ket 2 %} が得られます。 {% ket 2 %} 以外の確率はすべて 0% なので、これは重ね合わせ状態でありません。

{% include mixer_horizontal.svg class="h-4 w-4 text-gray-700" %}

次は {% ket 0 %}, {% ket 2 %}, {% ket 4 %}, {% ket 6 %} がそれぞれ確率 25% の重ね合わせ状態にあります。 3 量子ビットすべてを測定すると {% ket 0 %}, {% ket 2 %}, {% ket 4 %}, {% ket 6 %} のいずれかが確率 25% ずつ (つまり 1/4 の確率) で得られます。 重ね合わせではすべての確率を足すと必ず 100% になり、この場合もそうなっていることを確認してください。

{% include mixer_horizontal.svg class="h-4 w-4 text-gray-700" %}

最後はそれぞれの確率が異なり、さらに位相も異なる例です。 今までの例と同じく、それぞれの確率に応じて {% ket 0 %} 〜 {% ket 7 %} のいずれかが測定でき、すべての確率を足し合わせると 100% になります。 そして位相と確率はそれぞれ独立した量子ビットのプロパティなので、位相は確率にまったく影響しないことを思い出しておきましょう{% nmargin_note %}位相の意味や使い途については位相を参照してください。{% endnmargin_note %}。
