--- layout: page title: 量子 NOT ゲート (X ゲート) description: 量子ビットペアの入れ替え ---

量子 NOT ゲートは従来の NOT 演算に似た働きをします。 NOT 演算は、0 に適用するとビットが反転して 1 になり、逆に 1 に適用すると 0 になるのでした。つまり、NOT 演算は "0 と 1 を入れ替える" という働きをします。

  • NOT 0 {% include arrow_right.svg class="h-4 w-4 mx-2" %} 1
  • NOT 1 {% include arrow_right.svg class="h-4 w-4 mx-2" %} 0

量子 NOT ゲートは NOT 演算によく似ていて、量子ビットの {% include ket.html label="0" %} と {% include ket.html label="1" %} を入れ替えます。 図で表すと、次のような "円の入れ替え" の働きをします。

{% include minus.svg class="h-6 w-6 fill-current text-gray-400" %} {% include arrow_right.svg class="h-6 w-6 fill-current text-gray-400" %}
量子 NOT ゲートを回路に置いて、{% include ket.html label="0" %} と {% include ket.html label="1" %} が入れ替わることを確認しましょう。

量子 NOT ゲートが従来の NOT 演算と異なるのは、重ね合わせ状態にある量子ビットに作用できる点です。 たとえば、次の {% include ket.html label="0" %} と {% include ket.html label="1" %} の重ね合わせ状態にある量子ビットを考えましょう。

これに量子 NOT ゲートを適用すると、{% include ket.html label="0" %} と {% include ket.html label="1" %} の内容を交換します。 これはまさに、先ほど見た "{% include ket.html label="0" %} と {% include ket.html label="1" %} の入れ換え" という効果です。

{% include minus.svg class="h-6 w-6 fill-current text-gray-400" %} {% include arrow_right.svg class="h-6 w-6 fill-current text-gray-400" %}

量子 NOT ゲートを続けて 2 回適用するとどうなるでしょうか? 図から分かるように、 {% include ket.html label="0" %} と {% include ket.html label="1" %} が 2 度入れ替わることで元の状態に戻ります。 つまり、量子 NOT ゲートはそれ自身を打ち消す効果を持っています。数学ではこれを、「量子 NOT ゲートはそれ自身の逆演算である」と言います。

  • 重ね合わせ状態の量子ビットに量子 NOT ゲートを適用し、{% include ket.html label="0" %} と {% include ket.html label="1" %} が入れ替わることを確認しましょう。
  • 量子 NOT ゲートを 2 回適用すると元に戻ることを確認しましょう
{% include mixer_horizontal.svg class="h-4 w-4 text-gray-700" %}

量子 NOT ゲートはよく X ゲートとも呼ばれます。 その由来は、量子ビット状態をブロッホ球表すと量子 NOT ゲートは X 軸を中心とした回転操作の一種だからです。

同様に、Y 軸で回転する Y ゲート、Z 軸で回転する Z ゲートも存在します。さまざまなゲートの回転については、他のゲートを学んだ後で詳しく説明します。
