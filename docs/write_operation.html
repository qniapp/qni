--- layout: page title: WRITE 命令 description: 量子ビットの初期化 ---

WRITE 命令は量子ビットの状態を {% include ket.html label="0" %} または {% include ket.html label="1" %} に初期化する命令です。 実は今まで断りなく何度か使ってきましたが、使いかたは簡単です。 通常は回路の先頭に WRITE 命令を置くことで量子ビットを初期化してから、H ゲートなどの演算命令を置いていくことで量子プログラムを記述します。

量子ビットを {% include ket.html label="0" %} に初期化してからゲートを配置
量子ビットを {% include ket.html label="1" %} に初期化してからゲートを配置

WRITE 命令は量子ビットの状態を強制的にリセットするので、逆演算を持ちません。 量子ビットの状態がどんな状態にあっても、たとえば {% include ket.html label="0" %} 命令を置いた時点で振幅の大きさと位相の情報は破壊され、{% include ket.html label="0" %} に初期化されます。 これは右から左方向へは戻せない操作なので、WRITE 命令は破壊的であると言えます。
