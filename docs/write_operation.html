WRITE 命令

量子ビットの初期化

WRITE 命令は量子ビットの状態を |0⟩ または |1⟩ に初期化する命令です。 実は今まで断りなく何度か使ってきましたが、使いかたは簡単です。 通常は回路の先頭に WRITE 命令を置くことで量子ビットを |0⟩ や |1⟩ に初期化します。 続けて H ゲートなどの QPU 命令を置いていくことで量子プログラムを記述します。

量子ビットを |0⟩ に初期化してからゲートを配置
量子ビットを |1⟩ に初期化してからゲートを配置

WRITE 命令は量子ビットの状態を強制的にリセットするので、逆演算を持ちません。 つまり量子ビットの状態がどんな状態にあっても、たとえば |0⟩ 命令を置いた時点で振幅の大きさと位相の情報は破壊され、|0⟩ に初期化されます。 これは元に戻すのに必要な情報を損失していることになるため、WRITE 命令は非可逆です。
