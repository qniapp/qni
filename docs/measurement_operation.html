単純な状態の測定

MEASUREMENT (測定) 命令は量子ビットから 0 または 1 の値を読み出します。 量子ビット状態が {% include ket.html label="0" %} または {% include ket.html label="1" %} など重ね合わせ状態にない場合、測定するとそのまま {% include ket.html label="0" %} と {% include ket.html label="1" %} が返されます。

重ね合わせ状態の測定

量子ビットが重ね合わせ状態にある場合、0 と 1 が出る確率は {% include ket.html label="0" %} と {% include ket.html label="1" %} の確率 (円表示の青い部分の面積。振幅の大きさの 2 乗) に応じて決まります。 よって、結果にはランダム性があります。

75%

測定によって量子ビットの振幅の大きさと位相の情報が破壊されるので、MEASUREMENT 命令は WRITE 命令と同様に破壊的です。 また、位相と振幅の大きさは独立したパラメータなので、位相の値は測定結果にまったく何の影響も与えないことに注意しましょう。

量子乱数ジェネレータ

MEASUREMENT 命令とこれまでに学んだ WRITE 命令、H ゲートを組合わせることで、最初の量子アプリケーション「量子乱数ジェネレータ」を作ることができます。 WRITE で {% include ket.html label="0" %} に初期化した量子ビットを H ゲートで重ね合わせ状態にし、MEASUREMENT 命令で測定することで 0 または 1 をランダムに得ることができます。

重ね合わせ状態を測定して得られる量子乱数は、従来のコンピュータで計算的に生成する疑似乱数と異なり、原理的にまったく予測できない完全な乱数です。 その用途は思ったより広く、セキュリティ分野での暗号鍵の安全な生成はもちろん、モンテカルロ法など乱数を使ったアルゴリズムの速度や精度向上にもつながります。 実際に、量子乱数の生成はすでにサービス化されています。
